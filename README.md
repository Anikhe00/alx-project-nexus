# alx-project-nexus

The ALX ProDev Frontend Program is a program designed to help students master advanced front-end technologies, including React, Next.js, React Native, and TypeScript, to build responsive and dynamic user interfaces.

## Major Learnings
During the program, we focused on the following:

1. AI Prompting Framework, Requirement Analysis, and Understanding Our Learning Style.

2. Semantic HTML, CSS Grid and Flexbox, and Web Accessibility.

3. CSS Processors, Introduction to Typescript, and what a Work log is.

4. Scaffolding a Next.js project, Advanced Project Structuring, State Management.

5. Advanced TypeScript features in React.

6. State management with Redux and Context API in Typescript.

7. Leveraging Hooks and Custom Hooks in Typescript, Consuming RESTful APIs.

8. Introduction and Implementation of GraphQL.

9. Advanced Routing Techniques, Progressive Web APP fundamentals.

10. Introduction to Mobile Development, React Native.

11. Styling in React Native, State Management, and React Native Expo Router.

## Challenges Faced
Understanding GraphQL and Progressive Web App Development was challenging, but I was able to learn and improve through peer learning.

## Best Practices and Personal Takeaways

- Be open to learning
- Ask your mentors and fellow students questions when you need help.
- Don't stop building
